# Osmo — landing page

Know exactly what to say, to everyone who matters.

A static marketing site for **Osmo**: your relationship memory, the right words, the
right tone, the right moment — across every platform you already use.

## Stack

Plain HTML + CSS + JS. No build step, no dependencies.

- `index.html` — page markup, split into sections (header, hero, testimonial, feature stack, closing CTA, footer)
- `styles.css` — design tokens + all component styles, fully responsive
- `script.js` — rotating hero word, rotating testimonials, scroll-reveal

## Run locally

```bash
python3 -m http.server 8790
```

Then open http://localhost:8790
